## Simple Barcode Scanner with Cordova
Using [barcodescanner cordova plugin](http://plugins.cordova.io/#/package/com.phonegap.plugins.barcodescanner).  

To run this project:  
1. `cordova platform add android`  
2. `cordova run android`  

Supported codes:  

## Android
QR_CODE  
DATA_MATRIX  
UPC_E  
UPC_A  
EAN_8  
EAN_13  
CODE_128  
CODE_39  
CODE_93  
CODABAR  
ITF  
RSS14  
PDF417  
RSS_EXPANDED
